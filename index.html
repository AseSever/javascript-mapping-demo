JavaScript Mapping Demo

Making a basic web map with open source libraries and tools.

Demo objectives:

  • Display a map on a web page using HTML, CSS, and JavaScript only.
  • Add static data to the map.
  • Add dynamic content to the map.
  • Apply styling to the map elements (cartography).
  • Add controls to the map.
  • Add popups with custom content to the map.
